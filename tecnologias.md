# Tecnologías Utilizadas - Artesanos.com

Este documento detalla las tecnologías y librerías clave utilizadas en el desarrollo del proyecto.

## Backend
- **PHP 8.x**: Lenguaje de programación principal para la lógica del servidor.
- **PDO (PHP Data Objects)**: Para una comunicación segura con la base de datos (MySQL).
- **MySQL**: Motor de base de datos relacional.

## Frontend
- **HTML5 & CSS3**: Estructura y diseño (Vanilla CSS con variables modernas).
- **Bootstrap 5.3**: Framework para componentes y layout responsivo.
- **Google Fonts**: Fuentes 'Outfit' (encabezados) e 'Inter' (cuerpo).
- **Croppie.js**: Librería de JavaScript para el recorte y manipulación de imágenes antes de la subida.

## Herramientas de Desarrollo
- **Lucide Icons / Emoji**: Para la iconografía visual.
- **Finfo (PHP Extension)**: Para validación de tipos MIME de archivos subidos.
